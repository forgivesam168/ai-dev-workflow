Select the `Package contract` value from `Compact / Full`. Each declaration above appears exactly once. Without an external tracker, `Task/status SSOT` is an accessible package-relative or repository-relative file path. With an external tracker, use an identifiable URL, `owner/repo#number`, or current-repository Issue/PR number in both pointer fields; package files then retain only its pointer, decisions, and static evidence. Declare no second task/status owner elsewhere in the package. Filename existence does not mean an artifact or stage is complete.
